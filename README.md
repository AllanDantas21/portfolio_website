# Portfolio Website

Allan Dantas' modern portfolio website migrated to Angular, TypeScript, and Tailwind CSS.

## 🛠️ Technologies Used

- **Framework**: Angular 22 (with SSR)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animation**: Angular Animations
- **Internationalization**: @ngx-translate/core
- **Testing**: Vitest
- **Linting**: ESLint (angular-eslint)
